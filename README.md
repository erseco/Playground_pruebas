# Moodle Playground Assets

Archivos auxiliares para cargar una configuración personalizada del tema Adaptable en Moodle Playground.

## Archivos

- `adaptable-config.json`: configuración exportada del tema Adaptable.
- `apply-adaptable-config.php`: script PHP para importar la configuración en Moodle Playground.

## Uso

Estos archivos serán referenciados desde un blueprint JSON mediante URLs públicas de GitHub Raw.
